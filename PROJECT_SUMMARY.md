## 🎨 Customization Guide

### Change Colors
```css
/* In styles.css, find :root */
--primary-color: #0066cc;     /* Change this */
--secondary-color: #00b4d8;   /* And this */
```

### Update Contact Info
Search entire project for:
- `+919876543210` → Replace with your number
- `info@cncinstitute.com` → Replace with your email
- `New Delhi` → Replace with your city

### Add New Courses
1. Go to `courses.html`
2. Copy a `course-card` div
3. Update course details
4. Save and refresh

### Change Course Prices
Search for price values and update directly in HTML.

### Update Social Links
Find social icon links in:
- Top bar (index.html, all pages)
- Footer (all pages)
- Replace `#` with actual social URLs
