## 🚀 Customization Tips

### Change Colors
1. Open `styles.css`
2. Find `:root { ... }` at the top
3. Edit these variables:
   ```css
   --primary-color: #0066cc;
   --secondary-color: #00b4d8;
   ```

### Update Contact Details
1. Search for "+91 98765 43210" across all files
2. Replace with your phone number
3. Update email similarly

### Add New Courses
1. Go to `courses.html`
2. Copy a course-card div
3. Paste and update:
   - Course name
   - Icon (class)
   - Duration, level, price
   - Description
   - Features

### Change Text/Content
1. Open the HTML file in text editor
2. Find the text you want to change
3. Edit it directly
4. Save file
5. Refresh browser to see changes

## ❓ Troubleshooting

### Website won't open
- Make sure all files are in the same folder
- Try double-clicking index.html
- Try opening in different browser

### Styling looks wrong
- Clear browser cache (Ctrl+Shift+Delete)
- Press Ctrl+F5 to hard refresh
- Check that styles.css is in same folder

### Forms not working
- Check browser console (F12 → Console)
- Make sure JavaScript is enabled
- Try in different browser

### Images not showing
- Replace placeholder URLs with actual images
- Ensure image paths are correct
- Check image file permissions
