# CNC Institute - Networking & Cyber Security Training Website

A modern, premium, fully responsive website for a Networking & Cyber Security Training Institute featuring dark/light mode, professional UI/UX design, and comprehensive course offerings.

## 🌟 Features

### 🎨 Design & Styling
- **Modern SaaS Design** - Clean, professional interface
- **Premium Feel** - High-quality typography and spacing
- **Dark/Light Mode** - Theme switcher with user preference storage
- **Fully Responsive** - Mobile, tablet, laptop, and desktop optimized
- **Smooth Animations** - Subtle transitions and hover effects

### 📱 Navigation
- **Sticky Navbar** - Always accessible navigation
- **Services Mega Menu** - Professional services dropdown
- **Top Contact Bar** - Social media links and contact info
- **Mobile Menu** - Hamburger menu for mobile devices

### 🎯 Key Sections
- **Hero Section** - Eye-catching introduction with CTA buttons
- **Course Cards** - Showcase of 8+ training programs
- **Testimonials** - Student success stories with ratings
- **Image Gallery** - Student success gallery with lightbox
- **Placement Section** - Career assistance highlights
- **Footer** - Comprehensive link structure and information

### 📝 Forms
- **Registration Popup** - Auto-triggered registration form (2-second delay)
- **Demo Form** - Book free class section
- **Contact Form** - Get in touch with the institute

### 📄 Pages
- **Home** (index.html) - Landing page with all sections
- **Courses** (courses.html) - Detailed course listings
- **Blog** (blog.html) - Articles and insights
- **Victory** (victory.html) - Success stories and statistics
- **Contact** (contact.html) - Contact information and form
- **E-Certificate** (ecertificate.html) - Certificate verification portal

## 🚀 Getting Started

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required (static HTML/CSS/JS)

### Installation
1. Extract all files to your web server or local folder
2. Open `index.html` in your web browser
3. Explore all pages through the navigation menu

### File Structure
```
├── index.html              # Home page
├── courses.html            # Courses listing
├── blog.html               # Blog section
├── victory.html            # Success stories
├── contact.html            # Contact page
├── ecertificate.html       # Certificate portal
├── styles.css              # All styling (light/dark mode)
├── script.js               # JavaScript functionality
└── README.md               # This file
```

## 🔧 Features Breakdown

### Theme Switching
- Click moon/sun icon in navbar to toggle dark/light mode
- User preference automatically saved in browser

### Registration Popup
- Automatically appears 2 seconds after page load
- Can be closed manually
- Collects name, phone, email, and course interest

### Mega Menu
- Hover over "Services" in navbar to see professional dropdown

  - Networking Training (CCNA, CCNP, CCIE)

## 🛠️ Customization

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00b4d8;
    /* ... */
}
```

### Update Contact Info
Search for contact details across all HTML files and update:
- Phone numbers
- Email addresses
- Address
- Social media links

### Add More Courses
Copy the course-card div in `courses.html` and update:
- Course name
- Icon class
- Description
- Features list
- Price

### Modify Colors & Branding
All styling is in `styles.css`. Search for color values to customize the entire design.
